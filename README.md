# SegLab — Open-Source Brain MRI Lesion Segmentation, and Neuroanatomy Training
Brain lesion segmentation application on T1-weighted MRI with case loading, mask annotation, gold-mask comparison, and attempt-level scoring via Dice and IoU.

Open-source brain MRI lesion segmentation and neuroanatomy training platform with interactive annotation, expert-mask comparison, and instant Dice/IoU feedback for education, research, and clinical training.
